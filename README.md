# 📝 Blog Website

A clean and responsive **Blog Website** built using **HTML, CSS, and JavaScript**.
This project displays blog posts in a simple, readable layout with a modern UI. It is designed to be lightweight, fast, and easy to navigate.

🌐 **Live Website:**
https://ariz440.github.io/Blog-Website/

💻 **GitHub Repository:**
https://github.com/ariz440/Blog-Website

---

# 🚀 Features

* 📰 Clean blog layout
* 📱 Fully responsive design
* ⚡ Fast and lightweight website
* 🧭 Simple navigation
* 🎨 Modern user interface
* 🌍 Hosted using GitHub Pages

---

# 🛠️ Technologies Used

* **HTML5** – Website structure
* **CSS3** – Styling and layout
* **JavaScript** – Interactive elements
* **GitHub Pages** – Deployment and hosting

# ⚙️ Installation & Setup

To run this project locally:

### 1. Clone the repository

```
git clone https://github.com/ariz440/Blog-Website.git
```

### 2. Open the project folder

```
cd Blog-Website
```

### 3. Run the project

Open **index.html** in your browser.
